# Прогноз погоды

Vue 3 приложение, реализующее прототип прогноза погоды. В качестве данных используются mock-данные из файла.

## Запуск проекта

1.  Перейдите в директорию проекта:
    ```bash
    cd weather-app
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```
